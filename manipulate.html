Operations

numbro provides a set of basic operations to apply to numbers.

{% highlight javascript %}var number = numbro(1000); var added = number.add(10); // 1010{% endhighlight %}
Before Function After
1000 .add(100) 1100
1100 .subtract(100) 1000
1000 .multiply(100) 100000
100000 .divide(100) 1000

Value

The value of a numbro is always available via the value function.

{% highlight javascript %}var number = numbro(1000); var string = number.format('0,0'); // '1,000' var value = number.value(); // 1000{% endhighlight %}

Set

Set the value of a numbro.

{% highlight javascript %}var number = numbro(10); number.set(1000); var value = number.value(); // 1000{% endhighlight %}

Difference

Find the difference between your numbro object and a numeric value.

{% highlight javascript %}var number = numbro(1000), value = 100; var difference = number.difference(value); // 900{% endhighlight %}

Clone

Clone any numbro as easy as using clone.

{% highlight javascript %}var a = numbro(1000); var b = numbro(a); var c = a.clone(); var aVal = a.set(2000).value(); // 2000 var bVal = b.value(); // 1000 var cVal = c.add(10).value(); // 1010{% endhighlight %}
