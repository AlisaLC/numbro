Format

Numbers can be formatted to look like currency, percentages, times, or even plain old numbers with decimal places, thousands, and abbreviations.

{% highlight javascript %}var string = numbro(1000).format('0,0'); // '1,000'{% endhighlight %}

Currency

numbro supports cultural currency formatting via the function formatCurrency.

{% highlight javascript %}numbro(1000.234).formatCurrency(); // $1000{% endhighlight %}
Number Language String
1000.234 en-US
1000.234 fr-FR
1000.234 ja-JP
1000.234 ru-RU

You can of course precise a more specific format, as long as you do not use the $ (that would override defaults).

{% highlight javascript %}numbro(1000.234).formatCurrency("0[.]00"); // $1000.23{% endhighlight %}
Number Language String
1000.234 en-US
1000.234 fr-FR
1000.234 ja-JP
1000.234 ru-RU

But if you want to, you can always precise a format by hand to override the languages defaults.

Number Format String
1000.234 '$0,0.00'
1000.2 '0,0[.]00 $'
1001 '$ 0,0[.]00'
-1000.234 '($0,0)'
-1000.234 '$0.00'
1230974 '($ 0.00 a)'

Unformat

Got a formatted string? Use the unformat function to make it useful again.

{% highlight javascript %}var string = numbro().unformat('($10,000.00)'); // -10000{% endhighlight %}

Defaults

numbro allows you to the default format to use, as well as how to format 0.

Default format

Set a default format so you can use .format() without a string. The default format to '0,0'.

{% highlight javascript %}var number = numbro(1000); number.format(); // '1,000' numbro.setDefaults('$0,0.00'); number.format(); // '$1,000.00'{% endhighlight %}

Default zero

Set a custom output when formatting numbers with a value of 0.

{% highlight javascript %}var number = numbro(0); numbro.zeroFormat('N/A'); var zero = number.format('0.0') // 'N/A'{% endhighlight %}
